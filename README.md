## Overview

This is a **multi-page website** designed to showcase the role of a Cybersecurity Analyst. It highlights:

- Key responsibilities  
- Required skills and tools  
- Career path and certifications  
- Global cybersecurity job growth visualized with Chart.js  

The site is **fully responsive** and designed with readability and user experience in mind.

---

## Pages

1. **Home (`index.html`)** – Hero section, overview, and interactive job growth chart.  
2. **Responsibilities (`responsibilities.html`)** – Core duties and roles of a cybersecurity analyst.  
3. **Skills (`skills.html`)** – Technical, analytical, and incident management skills.  
4. **Career Path (`career.html`)** – Entry-level roles, certifications, and career growth.  

---

## Features

- **Responsive Layout** – Works on mobile, tablet, and desktop.  
- **Interactive Chart** – Line chart showing global cybersecurity job growth.  
- **Animated Sections** – Smooth fade-in animations using Animate.css.  
- **Navigation** – Sticky top menu with hover effects.  
- **Cards & Containers** – Clean, semi-transparent sections over a full-page background image.
